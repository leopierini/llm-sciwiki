# Rodriguez, Maria

**Type:** Person (Researcher)

## Overview

Neuroscientist who led the first RCT demonstrating that closed-loop auditory stimulation enhances spindle density and improves declarative memory (Rodriguez, 2024). Also co-authored the subsequent multi-site replication attempt (Kumar, 2025) that replicated the spindle enhancement but not the memory benefit.
