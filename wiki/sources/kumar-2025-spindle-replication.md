# Kumar (2025) — Closed-loop stimulation increases spindles but does not improve memory: multi-site replication

**Authors:** Kumar, Priya; Zhang, Liang; Fischer, Anna; Rodriguez, Maria
**Journal:** Psychological Science | **Year:** 2025
**DOI:** 10.1177/09567976251234567
**Type:** Paper | **Evidence Level:** 2 (RCT, multi-site) | **Peer Reviewed:** Yes

## Summary

Pre-registered, multi-site replication of Rodriguez (2024) across three laboratories (London, Tokyo, Toronto) with n=204 — more than double the original sample. Stimulation successfully replicated the physiological effect (24% spindle density increase, d=0.88), but did NOT replicate the memory benefit (d=0.18 for word-pairs vs original d=0.68; Bayes factor favoring null = 4.2). Exploratory subgroup analysis found that only participants with low baseline spindle density showed a memory benefit (d=0.52), suggesting a ceiling effect.

## Methods

- **Design:** Pre-registered, double-blind, sham-controlled, multi-site RCT
- **Sample:** 204 participants (age 20-38), 102 per group, across 3 sites
- **Key techniques:** [[methods/closed-loop-auditory-stimulation]], [[methods/polysomnography]], EEG, word-pair recall, face-name recall
- **Statistical approach:** Between-group comparison, Bayesian analysis, mediation analysis, subgroup analysis (exploratory)

## Key Findings

1. Spindle density increase replicated: 24% vs sham (d=0.88, p<0.001)
2. Memory benefit NOT replicated: word-pairs d=0.18 (p=0.22), face-names d=0.12 (p=0.41)
3. Bayesian evidence favored the null for memory: BF01=4.2 (word-pairs), 6.1 (face-names)
4. **Subgroup interaction (exploratory):** Low baseline spindle density → stimulation helped (d=0.52, p=0.03); high baseline → no effect (d=-0.06)
5. Mediation not significant in full sample (p=0.31); partial mediation in low-baseline subgroup only (38%, p=0.04)

## Claims Extracted

- [[claims/claim-spindle-enhancement-improves-memory]] — Contradicts. Memory benefit not replicated in adequately powered multi-site study (evidence level: 2, direction: contradicts)
- [[claims/claim-spindle-density-predicts-consolidation]] — Partially supports. Subgroup analysis suggests the relationship may be moderated by baseline spindle density

## Limitations

- Subgroup analysis was exploratory (not pre-registered)
- Different stimulation hardware across sites
- Young adults only (20-38)
- Only declarative memory tasks tested
- Original study author is co-author (reduces protocol concern, but potential bias)
