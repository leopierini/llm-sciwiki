# Rodriguez (2024) — Auditory closed-loop stimulation enhances sleep spindles and declarative memory

**Authors:** Rodriguez, Maria; Patel, Arjun; Hoffmann, Klaus
**Journal:** Nature Neuroscience | **Year:** 2024
**DOI:** 10.1038/s41593-024-01567-2
**Type:** Paper | **Evidence Level:** 2 (RCT) | **Peer Reviewed:** Yes

## Summary

Double-blind, sham-controlled RCT testing whether enhancing sleep spindles via auditory closed-loop stimulation improves declarative memory. Stimulation increased spindle density by 28% and improved next-morning word-pair recall. Mediation analysis confirmed spindles accounted for 62% of the memory benefit, providing causal evidence for the spindle-memory link.

## Methods

- **Design:** Double-blind, sham-controlled RCT
- **Sample:** 84 participants (age 22-40), 42 per group
- **Key techniques:** [[methods/closed-loop-auditory-stimulation]], [[methods/polysomnography]], EEG, word-pair recall
- **Statistical approach:** Between-group comparison, mediation analysis

## Key Findings

1. Stimulation increased spindle density by 28% vs sham (p<0.001, **d=0.95**)
2. Stimulation group showed better recall (78% vs 66%, p=0.003, **d=0.68**)
3. Spindle density increase mediated 62% of memory benefit (indirect effect p=0.008)
4. No differences in sleep architecture — effect specific to spindles

## Claims Extracted

- [[claims/claim-spindle-enhancement-improves-memory]] — Experimentally enhancing spindles causes improved memory consolidation (evidence level: 2, direction: supports)
- [[claims/claim-spindle-density-predicts-consolidation]] — Provides causal support via mediation analysis (evidence level: 2, direction: supports)

## Limitations

- Young adult sample (22-40)
- Single-night protocol — long-term effects unknown
- Word-pair task only
- Repeated stimulation effects not studied
