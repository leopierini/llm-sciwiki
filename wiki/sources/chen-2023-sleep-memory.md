# Chen (2023) — Sleep spindle density predicts memory consolidation in healthy adults

**Authors:** Chen, Wei; Nakamura, Yuki; Oberg, Sarah
**Journal:** Journal of Neuroscience | **Year:** 2023
**DOI:** 10.1523/JNEUROSCI.0142-23.2023
**Type:** Paper | **Evidence Level:** 3 (controlled observational) | **Peer Reviewed:** Yes

## Summary

Investigated whether sleep spindle density during a post-learning nap predicts declarative memory consolidation. Found that napping improved 24-hour word-pair recall compared to wakefulness, and that spindle density over centroparietal electrodes correlated with recall improvement within the nap group. The spindle-recall association was specific to consolidation (24h) and not present for immediate recall.

## Methods

- **Design:** Between-subjects (nap vs. wake), correlational within nap group
- **Sample:** 120 healthy adults (age 20-35), 60 per group
- **Key techniques:** [[methods/polysomnography]], EEG, word-pair recall task
- **Statistical approach:** Linear regression, correlation, controlling for age, baseline performance, total sleep time

## Key Findings

1. Nap group retained more word pairs at 24h than wake group (72% vs 58%, p<0.001, **d=0.81**)
2. Spindle density correlated with 24h recall improvement within nap group (r=0.43, p<0.001)
3. Effect survived covariate control (beta=0.38, p=0.002)
4. Spindle density did NOT predict immediate recall (r=0.08, p=0.54)

## Claims Extracted

- [[claims/claim-spindle-density-predicts-consolidation]] — Higher spindle density during post-learning sleep predicts better memory consolidation (evidence level: 3, direction: supports)

## Limitations

- Young adults only (20-35) — unclear if effects generalize to older adults where spindle activity declines
- Word-pair task only — one type of declarative memory
- Correlational within nap group — spindle density not experimentally manipulated
- Single nap session — stability unknown
