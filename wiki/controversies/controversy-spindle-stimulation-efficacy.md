# Controversy: Does closed-loop spindle stimulation actually improve memory?

## Summary

Closed-loop auditory stimulation reliably increases sleep spindle density (~25-28%), but whether this translates into a meaningful memory benefit is now contested. An initial RCT found a strong effect (d=0.68); a larger multi-site replication found a much smaller, non-significant effect (d=0.18, BF01=4.2).

## Position A: Spindle stimulation improves memory

**Key proponent:** [[entities/rodriguez-maria]] (Rodriguez, 2024)

**Evidence:**
- Single-site RCT (n=84): stimulation improved word-pair recall with d=0.68, p=0.003
- Mediation analysis confirmed 62% of memory benefit was mediated by spindle density increase
- Double-blind, sham-controlled design
- Source: [[sources/rodriguez-2024-spindle-stimulation]]

**Strengths:** Clean experimental design, mediation supports mechanism
**Weaknesses:** Single site, single study, relatively small sample (n=84)

## Position B: Spindle stimulation does not improve memory (or effect is much smaller)

**Key proponent:** [[entities/kumar-priya]] (Kumar, 2025)

**Evidence:**
- Pre-registered, multi-site RCT (n=204): d=0.18 (word-pairs), d=0.12 (face-names), both non-significant
- Bayesian analysis favored null hypothesis (BF01=4.2 for word-pairs)
- Mediation not significant in full sample (p=0.31)
- Spindle enhancement itself replicated perfectly (d=0.88)
- Source: [[sources/kumar-2025-spindle-replication]]

**Strengths:** Pre-registered, multi-site, 2.4x larger sample, Bayesian analysis, two memory tasks
**Weaknesses:** Different hardware across sites (though protocols matched)

## Possible Resolution: Baseline Moderator Hypothesis

Kumar (2025) found an exploratory interaction: stimulation only improved memory in participants with low baseline spindle density (d=0.52), not in those with average/high baseline density (d=-0.06). If confirmed, this suggests:

1. The original Rodriguez sample may have coincidentally included more low-baseline participants
2. The relationship between spindles and memory is not linear — there may be a sufficiency threshold
3. Clinical applications should target individuals with suboptimal spindle activity

**Status:** This moderator hypothesis is exploratory and not pre-registered. It requires confirmatory testing.

## Current Assessment

The overall memory benefit of spindle stimulation should be considered **contested**. The physiological effect (spindle enhancement) is robust and replicated. The cognitive benefit may depend on individual differences in baseline spindle activity — a possibility that has shifted the research question from "Does spindle stimulation improve memory?" to "For whom does spindle stimulation improve memory?"

## What Would Resolve This

1. A pre-registered RCT specifically testing the baseline spindle density moderator
2. A meta-analysis pooling individual participant data from both studies
3. Studies in populations with known low spindle density (e.g., older adults, insomnia patients)
