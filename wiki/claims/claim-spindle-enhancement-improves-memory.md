# Claim: Experimentally enhancing sleep spindles via closed-loop auditory stimulation improves declarative memory consolidation

**Confidence:** Contested
**Evidence Level:** 2 (RCTs, conflicting results)
**Replication Status:** Failed replication (memory benefit not replicated; physiological effect replicated)

## Evidence Summary

An initial RCT (Rodriguez, 2024, n=84) found that closed-loop auditory stimulation improved word-pair recall (d=0.68). A larger pre-registered multi-site replication (Kumar, 2025, n=204) replicated the spindle enhancement but NOT the memory benefit (d=0.18, BF01=4.2 favoring the null). Exploratory subgroup analysis suggests the effect may be limited to individuals with low baseline spindle density (d=0.52).

## Effect Size Range

- **Range:** d = 0.18 - 0.68
- **Typical:** Uncertain — possibly near-zero for general population, moderate (d=0.52) for low-baseline subgroup
- **Heterogeneity:** High (original and replication diverge substantially)

## Subgroup Findings (Exploratory)

Kumar (2025) found a significant baseline spindle density × stimulation interaction:
- **Low baseline spindle density:** d=0.52 (p=0.03) — benefit observed
- **High baseline spindle density:** d=-0.06 (p=0.81) — no benefit

This suggests a ceiling effect but is **exploratory and not pre-registered**. See [[controversies/controversy-spindle-stimulation-efficacy]].
