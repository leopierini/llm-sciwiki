# Comparison: Rodriguez (2024) vs Kumar (2025) Stimulation Protocols

## Overview

Head-to-head comparison of the two closed-loop auditory stimulation studies that produced conflicting memory outcomes.

## Protocol Comparison

| Feature | Rodriguez (2024) | Kumar (2025) |
|---------|-----------------|--------------|
| **Design** | Double-blind, sham-controlled RCT | Pre-registered, double-blind, sham-controlled, multi-site RCT |
| **Sample size** | n=84 (42/group) | n=204 (102/group) |
| **Sites** | 1 | 3 (London, Tokyo, Toronto) |
| **Age range** | 22-40 | 20-38 |
| **Stimulation type** | Auditory clicks timed to SO up-phase | Auditory clicks timed to SO up-phase |
| **Hardware** | Single system | Matched protocol, different hardware per site |
| **Memory tasks** | Word-pair recall | Word-pair recall + face-name recall |
| **Sleep protocol** | Overnight | Overnight |
| **Pre-registered** | Not stated | Yes |
| **Bayesian analysis** | No | Yes |

## Outcome Comparison

| Outcome | Rodriguez (2024) | Kumar (2025) |
|---------|-----------------|--------------|
| Spindle density increase | 28% (d=0.95) | 24% (d=0.88) |
| Word-pair recall (stim vs sham) | 78% vs 66% (d=0.68, p=0.003) | 68% vs 65% (d=0.18, p=0.22) |
| Face-name recall | Not tested | 71% vs 69% (d=0.12, p=0.41) |
| Mediation significant? | Yes (62%, p=0.008) | No (full sample p=0.31) |
| Baseline moderator tested? | No | Yes (exploratory) |

## Key Differences That May Explain Divergence

1. **Statistical power:** Rodriguez was powered to detect d=0.6+; Kumar was powered to detect d=0.4+. Kumar's null finding is more informative.
2. **Site heterogeneity:** Multi-site designs introduce variability but improve generalizability.
3. **Hardware variation:** Different stimulation systems across Kumar's sites may have introduced noise.
4. **Baseline spindle density:** Not reported in Rodriguez; Kumar's subgroup analysis suggests it moderates the effect.
5. **Sham group performance:** Rodriguez's sham group performed lower (66%) than Kumar's (65-69%), suggesting possible sample differences.

## Assessment

The physiological effect (spindle enhancement) is consistent across both studies. The memory effect is not. The most parsimonious explanation is that the true memory effect is smaller than Rodriguez's initial estimate, possibly moderated by baseline spindle density.
